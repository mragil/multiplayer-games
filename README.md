# multiplayer-games-monorepo

Multiplayer games, made with Sveltekit on the frontend and Bun on the backend.

To install dependencies:

```bash
bun install
```

To run backend:
```bash
cd packages/backend
bun src/index.ts
```

To run frontend:

```bash
cd packages/frontend
bun run dev
```

This project was created using `bun init` in bun v1.2.13. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
